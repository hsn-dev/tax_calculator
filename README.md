# Tax Calculator

### Setup

curl -fsSL https://github.com/rbenv/rbenv-installer/raw/HEAD/bin/rbenv-installer | bash

`echo 'export PATH="$HOME/.rbenv/bin:$PATH"' >> ~/.bashrc`

`echo 'eval "$(rbenv init -)"' >> ~/.bashrc`

`source ~/.bashrc`

`rbenv install 3.3.6`

`gem install rspec`

`rspec tax_processor_tests.rb`
